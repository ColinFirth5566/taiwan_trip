# 🗺️ 台灣嘉義台南旅遊路線

一個互動式的旅遊路線規劃網站，使用 Three.js 3D 視覺化和 Google Maps 展示台灣嘉義和台南的精選旅遊路線。

## 🌟 特色功能

- **3D 路徑視覺化**：使用 Three.js 創建動畫 3D 路徑展示
- **Google Maps 整合**：顯示實際地圖路線和景點
- **兩條精選路線**：
  - 路線二：嘉義文化之旅
  - 第一天：台南美食之旅
- **響應式設計**：完美支援手機、平板和電腦
- **互動式卡片**：詳細的景點資訊和時間規劃

## 🛠️ 技術棧

- **Three.js** - 3D 圖形渲染
- **Google Maps API** - 地圖整合
- **Tailwind CSS** - UI 設計
- **Vanilla JavaScript** - 無框架純 JS

## 📝 使用說明

1. 點擊上方按鈕切換不同路線
2. 3D 視覺化會自動旋轉展示景點
3. 點擊 Google Maps 上的標記查看景點詳情
4. 查看景點卡片了解更多資訊

## 🔧 本地開發

```bash
# 克隆專案
git clone https://github.com/ColinFirth5566/taiwan_trip.git

# 進入目錄
cd taiwan_trip

# 使用任何 HTTP 伺服器開啟 index.html
# 例如使用 Python:
python -m http.server 8000

# 或使用 Node.js:
npx http-server
```

然後在瀏覽器打開 `http://localhost:8000`
